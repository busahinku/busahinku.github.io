## Overview

This portfolio website showcases my work and writings. Built with modern web technologies, it features a clean design with multiple theme options.

## Features

- **Multi-language Support**: English and Turkish translations
- **Theme System**: Light, Dark, and Paper themes
- **Blog System**: Full-featured blog with categories, tags, and search
- **Responsive Design**: Mobile-first approach
- **Performance**: Static site generation for fast loading

## Tech Stack

- Astro 5.x
- TypeScript
- CSS Variables for theming
- Firebase for analytics and comments

## Lessons Learned

Building this project taught me about static site generation, content collections in Astro, and implementing internationalization without external libraries.
