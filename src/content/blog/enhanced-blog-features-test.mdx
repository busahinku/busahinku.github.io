## Mathematical Equations with KaTeX

One of the most powerful features is full LaTeX math support<Cite id="knuth" num={1} />. You can write inline math like $E = mc^2$ or the quadratic formula $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ directly in your text.

For more complex equations, use display mode:

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

Here's the famous Euler's identity:

$$
e^{i\pi} + 1 = 0
$$

And a matrix example:

$$
\begin{bmatrix}
a & b \\
c & d
\end{bmatrix}
\begin{bmatrix}
x \\
y
\end{bmatrix}
=
\begin{bmatrix}
ax + by \\
cx + dy
\end{bmatrix}
$$

### Calculus Example

The derivative of a function using the limit definition:

$$
f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h}
$$

## Interactive Concept Checks

Test your understanding with interactive questions. Click on the answer area to reveal the solution!

<ConceptCheck question="What is the time complexity of binary search?">
  **O(log n)** - The search space is halved with each iteration, leading to logarithmic time complexity<Cite id="cormen" num={2} />. For an array of 1 million elements, binary search needs at most 20 comparisons.
</ConceptCheck>

<ConceptCheck question="What does the Big O notation O(n) mean?">
  **Linear time complexity** - The algorithm's running time grows linearly with the input size. If you double the input, the time roughly doubles as well.
</ConceptCheck>

<ConceptCheck>
  <span slot="question">In the equation $ax^2 + bx + c = 0$, what is the discriminant?</span>
  The discriminant is $\Delta = b^2 - 4ac$. It determines the nature of the roots:
  - If $\Delta > 0$: Two distinct real roots
  - If $\Delta = 0$: One repeated real root
  - If $\Delta < 0$: Two complex conjugate roots
</ConceptCheck>

### Code Blocks

```javascript
function binarySearch(arr, target) {
  let left = 0;
  let right = arr.length - 1;

  while (left <= right) {
    const mid = Math.floor((left + right) / 2);

    if (arr[mid] === target) {
      return mid;
    } else if (arr[mid] < target) {
      left = mid + 1;
    } else {
      right = mid - 1;
    }
  }

  return -1;
}
```
